# YouTube-Downloader
Расширение для Google Chrome и Яндекс Браузера, которое добавляет на странице видео YouTube кнопку «Скачать». По нажатию открывается меню с выбором качества и примерным весом файла. Само скачивание выполняет локальный сервер на базе `yt-dlp` — он умеет качать любое качество и склеивать видео со звуком.
