# YT Downloader — кнопка скачивания на YouTube

Расширение для **Google Chrome** и **Яндекс Браузера**, которое добавляет
на странице видео YouTube кнопку **«Скачать»**. По нажатию открывается
меню с выбором качества и размера файла. Само скачивание выполняет
локальный сервер на базе `yt-dlp`, а лаунчер с иконкой в трее следит,
чтобы сервер был запущен.

```text
yt-downloader/
├── YT Downloader.exe      ← только в собранном релизе
├── server/
│   ├── server.py
│   ├── server.exe         ← только в собранном релизе
│   └── requirements.txt
└── extension/
    ├── manifest.json
    ├── content.js
    ├── content.css
    ├── background.js
    ├── popup.html
    ├── popup.js
    └── icons/
```

## Почему гибрид: расширение + сервер

Обычное браузерное расширение не умеет надёжно скачивать YouTube в
хорошем качестве, потому что:
- видео и аудио часто идут раздельными потоками;
- для склейки нужен `ffmpeg`;
- ссылки и форматы YouTube постоянно меняются.

Поэтому интерфейс даёт расширение, а реальную загрузку делает локальный
сервер с `yt-dlp`, которым управляет лаунчер (иконка в трее).

## Способ 1 — готовая программа (.exe), рекомендуется

Ничего устанавливать не нужно, кроме `ffmpeg`.

1. Скачай последний архив со страницы **[Releases](../../releases)**
   (`YT-Downloader-vX.X.X-win64.zip`) и распакуй в любую папку.
2. Убедись, что установлен `ffmpeg` (см. раздел [ffmpeg](#ffmpeg) ниже).
3. Запусти `YT Downloader.exe`.
   - Появится окно с кнопкой **Run** — она сама станет красной и
     покажет **Stop**, когда сервер запустится.
   - Окно можно закрыть — программа останется в трее рядом с часами.
4. Установи расширение в браузер — см. [раздел ниже](#установка-расширения),
   указав папку `extension` из распакованного архива.
5. Всё готово — открывай YouTube и жми **«Скачать»**.

### Автозапуск

В окне программы есть переключатели:
- **Запускать сервер при старте программы** — включён по умолчанию.
- **Запускать при включении Windows** — программа будет тихо стартовать
  в трее вместе с Windows, ничего запускать вручную не придётся.

После изменения переключателей нажми **Сохранить**.

## Способ 2 — запуск из исходников (для разработки)

Нужен **Python 3.9+** (при установке отметь **Add Python to PATH**).

```bash
pip install -r requirements-launcher.txt
pip install -r server/requirements.txt

python launcher.py
```

Лаунчер сам запустит `server/server.py` через тот же интерпретатор
Python, поскольку рядом ещё нет `server.exe`.

Хочешь без лаунчера — можно запускать сервер напрямую:
```bash
python server/server.py
```

## ffmpeg

`ffmpeg` нужен для склейки видео и аудио в один файл и не входит ни в
`server.exe`, ни в исходники — его нужно поставить отдельно один раз.

**Через winget (Windows 10/11):**
```bash
winget install Gyan.FFmpeg
```

**Вручную:** скачай сборку с [gyan.dev](https://www.gyan.dev/ffmpeg/builds/)
и добавь папку `bin` в переменную окружения `PATH`.

Проверить, что всё встало: открой командную строку и выполни `ffmpeg -version`.

## Проверка сервера

Пока сервер запущен, в браузере должен открываться:
- `http://127.0.0.1:5001/ping`

Если всё хорошо — вернётся JSON с `"ok": true`.

Сервер слушает только `127.0.0.1`, то есть доступен исключительно
локально на твоём ПК.

## Установка расширения

### Google Chrome
1. Открой `chrome://extensions`
2. Включи **Режим разработчика**
3. Нажми **Загрузить распакованное расширение**
4. Выбери папку `extension`

### Яндекс Браузер
1. Открой `browser://extensions`
2. Включи **Режим разработчика**
3. Нажми **Загрузить распакованное расширение**
4. Выбери папку `extension`

---

## Использование

1. Запусти `YT Downloader.exe` (или `python launcher.py`) — дождись, пока
   кнопка в окне станет красной и покажет **Stop**.
2. Открой видео или Shorts на YouTube.
3. Нажми кнопку **Скачать** рядом с лайком.
4. Выбери качество из списка.
5. Дождись завершения — файл появится в загрузках браузера (`Ctrl+J`).

Файлы также сохраняются на диске в:
- `Downloads/YouTube`

## Частые проблемы

| Проблема | Решение |
|---|---|
| В окне программы горит «Офлайн» | Нажми **Run** — сервер ещё не запущен |
| Кнопка «Скачать» не появляется на YouTube | Перезагрузи страницу, проверь, что расширение включено |
| Ошибка при склейке видео | Не установлен `ffmpeg` — см. раздел [ffmpeg](#ffmpeg) |
| `server.py` не запускается из исходников | Проверь, что выполнил `pip install -r server/requirements.txt` |

---

## Безопасность

Сервер и расширение работают полностью локально: сервер слушает только
`127.0.0.1` и недоступен из интернета или локальной сети.
