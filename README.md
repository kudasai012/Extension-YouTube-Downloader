# YT Downloader — кнопка скачивания на YouTube

Расширение для **Google Chrome** и **Яндекс Браузера**, которое добавляет на странице видео
YouTube кнопку **«Скачать»** (слева от лайка). По нажатию открывается меню с выбором
качества и **примерным весом** файла. Само скачивание выполняет локальный сервер на
базе `yt-dlp` — он умеет качать любое качество и склеивать видео со звуком.

```
yt-downloader/
├── server/        ← локальный сервер (запускается на твоём ПК)
│   ├── server.py
│   ├── requirements.txt
│   └── start_server.bat
└── extension/     ← само расширение для браузера
    ├── manifest.json
    ├── content.js / content.css
    ├── background.js
    ├── popup.html / popup.js
    └── icons/
```

## Почему гибрид (расширение + сервер)?

Браузерное расширение **не может** само скачать видео с YouTube корректно: потоки видео и
аудио отдаются раздельно (DASH) и зашифрованы, склейка требует ffmpeg. Всё это умеет
`yt-dlp` — поэтому тяжёлую работу делает локальный сервер, а расширение — это удобный UI.

---

## Шаг 1. Установить сервер

Нужны **Python 3.9+** и **ffmpeg**.

1. Установи Python (отметь галочку *Add Python to PATH*).
2. Установи ffmpeg и добавь его в PATH
   (Windows-сборки: https://www.gyan.dev/ffmpeg/builds/).
3. Зайди в папку `server` и запусти **`start_server.bat`**.
   Он сам поставит зависимости и поднимет сервер на `http://127.0.0.1:5001`.

> Видео сохраняются в `Загрузки/YouTube` (`Downloads/YouTube`).

Проверка: открой в браузере `http://127.0.0.1:5001/ping` — должно вернуть `{"ok": true, ...}`.

---

## Шаг 2. Установить расширение

### Google Chrome
1. Открой `chrome://extensions`.
2. Включи **«Режим разработчика»** (Developer mode) справа сверху.
3. Нажми **«Загрузить распакованное расширение»** (Load unpacked).
4. Выбери папку **`extension`**.

### Яндекс Браузер
1. Открой `browser://extensions`.
2. Включи **«Режим разработчика»**.
3. **«Загрузить распакованное расширение»** → выбери папку **`extension`**.

---

## Использование

1. Запусти сервер (`start_server.bat`).
2. Открой любое видео на YouTube.
3. Слева от кнопки лайка появится **«Скачать»** (выглядит как нативные кнопки YouTube).
4. Нажми → выбери качество (минимум **360p**, рядом показан примерный вес).
5. В меню расширения показывается прогресс скачивания. Файл сохраняется
   на ПК в папку **Downloads/YouTube**.
6. По завершении — кнопка **📂 Открыть папку** откроет проводник с готовым файлом.

> Скачивание выполняет локальный сервер (yt-dlp), файл сразу попадает в
> `Downloads/YouTube`. В список загрузок браузера он не дублируется — открывай
> папку кнопкой 📂.

## Можно ли вообще без сервера?
Нет. YouTube отдаёт видео и аудио раздельно (DASH/SABR) и шифрует ссылки на потоки;
склейка требует ffmpeg, а расшифровка постоянно меняется. Поэтому даже коммерческие
расширения для 1080p/4K используют десктопное приложение-компаньон (по сути — такой же
локальный сервер). Чистое расширение может качать максимум «прогрессивные» потоки
(обычно до 360–720p со звуком) и работает нестабильно. Наш гибрид — самый надёжный путь.

---

## Безопасность
Сервер слушает только `127.0.0.1` (локально), наружу не виден. Используй для личных целей.
