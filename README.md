# YT Downloader — кнопка скачивания на YouTube

Расширение для **Google Chrome** и **Яндекс Браузера**, которое добавляет на странице видео YouTube кнопку **«Скачать»**. По нажатию открывается меню с выбором качества и размера файла. Скачивание выполняет локальный сервер на базе `yt-dlp`.

```text
yt-downloader/
├── server/
│   ├── server.py
│   ├── requirements.txt
│   └── start_server.bat
└── extension/
    ├── manifest.json
    ├── content.js
    ├── content.css
    ├── background.js
    ├── popup.html
    ├── popup.js
    └── icons/
```

## Почему гибрид: расширение + сервер

Обычное браузерное расширение не умеет надёжно скачивать YouTube в хорошем качестве, потому что:
- видео и аудио часто идут раздельными потоками;
- для склейки нужен `ffmpeg`;
- ссылки и форматы YouTube постоянно меняются.

Поэтому интерфейс даёт расширение, а реальную загрузку делает локальный сервер с `yt-dlp`.

---

## Быстрый старт

### 1. Запуск сервера

Открой:
- `yt-downloader/server/start_server.bat`

Что делает bat-файл:
- ищет `Python`
- пытается найти `ffmpeg`
- если `ffmpeg` не найден — пытается установить его через `winget`
- ставит Python-зависимости
- запускает сервер на `http://127.0.0.1:5001`

### Важное ограничение

#### Python
Если у пользователя **вообще нет Python**, bat-файл не сможет magically запустить сервер без него.
Для текущей версии проекта Python **обязателен**.

То есть сейчас нужно:
- установить **Python 3.9+**
- желательно отметить **Add Python to PATH**

#### ffmpeg
Если в системе есть `winget`, bat-файл попробует установить:
- `Gyan.FFmpeg`

Если `winget` отсутствует или установка не удалась, нужно установить `ffmpeg` вручную.

---

## Фоновый запуск

Можно запустить:
- `start_server.bat 1`

Тогда bat попробует открыть сервер в **свёрнутом окне**.

### Важно
Полностью спрятать обычный `.bat` в системный трей **невозможно одним bat-файлом**.
Для трей-режима нужен отдельный launcher / exe-приложение.

То есть сейчас доступно:
- обычный запуск: `start_server.bat`
- свёрнутый запуск: `start_server.bat 1`

---

## Проверка сервера

Открой в браузере:
- `http://127.0.0.1:5001/ping`

Если всё хорошо, вернётся JSON с `ok: true`.

---

## Установка расширения

### Google Chrome
1. Открой `chrome://extensions`
2. Включи **Режим разработчика**
3. Нажми **Загрузить распакованное расширение**
4. Выбери папку `yt-downloader/extension`

### Яндекс Браузер
1. Открой `browser://extensions`
2. Включи **Режим разработчика**
3. Нажми **Загрузить распакованное расширение**
4. Выбери папку `yt-downloader/extension`

---

## Использование

1. Запусти `start_server.bat`
2. Открой видео на YouTube
3. Нажми кнопку **Скачать**
4. Выбери качество
5. Дождись завершения загрузки
6. Нажми **📂 Открыть папку**

Файлы сохраняются в:
- `Downloads/YouTube`

---

## Можно ли без сервера?

Нет. Для стабильной загрузки YouTube-видео в хорошем качестве нужен `yt-dlp` и `ffmpeg`, поэтому без локального сервера этот проект работать надёжно не будет.

---

## Безопасность

Сервер слушает только:
- `127.0.0.1`

То есть доступен только локально на твоём ПК.
